The fp rule makes use of an assumption of a loop ⟨a*⟩P to read off a fixpoint J of ⟨a⟩ that is implied by postcondition P:

  1. Use fixpoint: J can be assumed to hold
  2. Fixpoint: J follows whenever P or when J holds after ⟨a⟩.
Make sure the fixpoint J is true before running a if there is a way of making J true after running a.
