gEx proves global existence of solutions for ODE x'=f(x), an important underlying assumption for ODE liveness proofs.

Invoking gEx will automatically attempt to find a proof. If the automation fails, an error message will ask for appropriate user hint input to gEx.

Global existence premises are automatically discharged (where possible) by diamond modality ODE tactics.

See also

  • dV Proves diamond modality questions for ODEs and may sometimes generate global existence premises.
