The loop rule proves safety of a loop by a given loop invariant J, which:

  1. initial case: needs to be true initially
  2. use case: needs to imply the postcondition P
  3. induction step: is still true after one round of the loop whenever it was true before
When identifying loop invariants, it is important that it transports enough knowledge about how the state might have changed during past iterations of a*.
