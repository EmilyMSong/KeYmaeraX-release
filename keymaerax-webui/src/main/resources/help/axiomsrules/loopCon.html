The con rule proves liveness of a loop ⟨a*⟩P by a loop convergence condition J(n):

  1. initial case: ∃n J(n) holds for some abstract distance n
  2. use case: n≤0, J(n) implies the postcondition P
  3. induction step: there is a way of running one iteration a to make J(n-1) true whenever n>0, J(n) was true before
