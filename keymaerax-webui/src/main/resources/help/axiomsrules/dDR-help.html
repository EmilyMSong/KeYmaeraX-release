dDR replaces domain constraint Q in ⟨x'=f(x)&Q⟩P with formula R by showing that the ODE always stays in domain Q within the larger domain R.

A useful choice for R is the formula true.

See also

  • dV The left premise (diamond modality) of dDR with domain constraint R can be proved using dV.
  • dI The right premise (box modality) of dDR can be proved using ODE proof rules for the box modality.
