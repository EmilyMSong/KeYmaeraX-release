Quantifier elimination QE decides truth of real arithmetic formulas.

QE may potentially take a long time. Reducing the number of variables that occur in formulas helps. Use Edit → abbrv to simplify formulas together with hideL and hideR to hide irrelevant assumptions and alternatives prior to QE.

See also

  • Edit for formula editing and abbreviations.
  • Tools → Smart Arithmetic for quantifier elimination that attempts automated hiding.
  • Tools → Counterexample to get potentially faster feedback whether the sequent might be true or false.
  • Tools → Find assumptions to get hints on how to strengthen the assumptions to make the sequent provable.
