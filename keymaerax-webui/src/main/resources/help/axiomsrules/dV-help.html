dV proves formula P to be a differential variant, i.e., ⟨x'=f(x)⟩P, by proving that ODE x'=f(x) makes ε progress towards P.

The formula P can be an atomic inequality like e >= 0 or a more complicated formula involving conjunctions and disjunctions of inequalities.

dV generates (P)' >= ε where the existentially quantified variable ε > 0 can be instantiated; ε can also be directly provided as input to dV.
