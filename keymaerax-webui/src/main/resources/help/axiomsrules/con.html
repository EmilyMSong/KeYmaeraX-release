The con rule proves liveness of a loop ⟨a*⟩P by a loop convergence condition J(n):

  1. initial case: ∃n J(n) is true initially
  2. use case: n≤0, J(n) implies the postcondition P
  3. induction step: J(n-1) is true after one round of the loop whenever n>0, J(n) was true before
